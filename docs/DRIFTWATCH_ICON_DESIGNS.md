# DriftWatch — Icon Design Prompts

Use these prompts in any AI image generator (DALL-E, Midjourney, Ideogram, etc.) to create a DriftWatch app icon/logo.

## Prompt 1 — Radar Sweep Shield

> A modern flat app icon for a developer tool called "DriftWatch". The icon is a dark navy (#0F172A) rounded square with a glowing radar sweep in electric purple-blue (#605DFF). The radar shows concentric rings radiating outward from a central shield shape. Small colored dots (red #EF4444, amber #F59E0B, green #10B981) are scattered on the radar rings representing risk levels. The sweep arm glows with a trailing gradient. Minimal, clean, professional. No text. Suitable for a 512x512 app icon.

## Prompt 2 — Abstract Drift Wave

> A sleek minimalist app icon for "DriftWatch", a DevOps risk intelligence tool. Dark navy (#0F172A) background with a stylized ocean wave made of gradient lines flowing from left to right, transitioning from green (#10B981) through amber (#F59E0B) to red (#EF4444). Above the wave, a single glowing eye/lens in electric blue (#605DFF) watches over it. The wave represents "drift" in code deployments. Clean vector style, no text, rounded corners. 512x512.

## Prompt 3 — Concentric Blast Radius

> A bold geometric app icon for a code analysis tool. Deep navy (#0F172A) rounded square background. In the center, concentric circles expand outward like a blast radius: innermost ring is bright blue-purple (#605DFF), second ring is red (#EF4444), third ring is amber (#F59E0B), outermost ring fades to teal (#06B6D4). A small diamond/hexagon sits at the center representing a code commit. Subtle grid lines in the background. Flat design, no gradients except glow effects, no text. App icon format 512x512.

## Prompt 4 — Guardian Sentinel Eye

> An app icon for "DriftWatch" — an AI-powered deployment guardian. Dark background (#0F172A). A stylized vertical eye shape made of two curved brackets `{ }` in white, with a glowing iris in the center that uses a gradient from purple (#9333EA) to electric blue (#605DFF). Below the eye, a subtle waveform/heartbeat line pulses in green (#10B981). The overall feel is "AI watching over your deployments". Modern, minimal, tech-forward. No text. 512x512 rounded square.

## Prompt 5 — Compass Navigator

> A premium app icon for a DevOps navigation tool. Navy (#0F172A) rounded square. A modern compass rose in the center with four cardinal points. The north point glows green (#10B981) for "safe to deploy", east is blue (#605DFF) for "analyzing", south is red (#EF4444) for "blocked", west is amber (#F59E0B) for "caution". The compass needle points north (safe). Subtle concentric circles behind the compass. A small satellite orbit ring around the outside in teal (#06B6D4). Clean flat vector. No text. 512x512.

## Usage Notes

- Generate at **512x512** minimum for app icons
- For favicon: export at 32x32 and 16x16
- For Apple Touch: 180x180
- For social/OG images: generate a wider landscape version (1200x630)
- The navy `#0F172A` background works well on both light and dark app docks
